# Current features

* Loading
  * CSV
* Measures - all measures can be configured to treat graphs as directed and undirected
  *  Closeness
  *  Local clustering
  *  Eigenvector
  *  Hits
  *  Neighbor connectivity
  *  Vetex embeddedness
* Experiments
  *  Describe graph using all measures to CSV files
